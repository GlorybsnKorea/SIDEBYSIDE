📖 HOW TO USE — FLAT BARGE SIDE-BY-SIDE TOWING SIMULATOR
STEP 1
Select a PRESET from the bar above the simulation, or manually drag the S (STBD) and P (PORT) sliders in the Propeller Control panel on the left.
STEP 2
Press ▶ START to run the simulation. The combined barge+tug moves according to thrust. Press ↺ RESET to return to start.
STEP 3
Watch the thrust arrows at the tug stern. Arrow direction = actual propeller thrust vector. Degree value shown on each arrow. Green = ahead, Red = astern.
STEP 4
Monitor YAW RATE in the control panel. If turning, adjust S or P to balance. For straight-ahead towing target YAW = 0°/s.
STEP 5
Use Combination tab for combined handle control. Set both levers AH 90° then adjust Combination Handle toward STBD 10~13° for straight towing.
⚠ CAUTION
During high-speed towing, use only Steering Lever AS for emergency stops (S: AS 90° / P: AS 30~60°). Do NOT use Combination Handle at high speed — risk of rope snap & bow collision with barge.
